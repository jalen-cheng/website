EasyASL

Hackathon winner accomplished within 36 hours. EasyASL provides a comprehensive tool for the hard-of-hearing community and other individuals who use American Sign Language (ASL). EasyASL makes it easy for you to learn with a reference index, translate in real time, and get certified in a secure database via randomized testing.

  • TypeScript
  • React

IFDAA (in progress)

Personal project completed for ECE198 course. The Intelligent Fall Detection Alert Apparatus (IFDAA) provides a robust, cost-effective solution for at-risk individuals. As an AIO patient care device, it can successfully detect, communicate and log falls for optimal wellbeing. Developed using 2 STM32F401RE Nucleo boards and a custom-modelled case.

  • C++
  • AutoDesk Fusion
  • draw.io

ecoswitch

Hackathon project completed in under 24 hours. ecoswitch serves as a flexible universal webapp to scan any UPC barcode item, and determine its sustainability with 3 metrics. It also has the ability to webscrape for better alternatives, grade a company on monetized environmental impact, and parse/summarize relevant articles.

  • TypeScript
  • Node.JS
  • React
  • Unwrangle API
  • OpenAI API
